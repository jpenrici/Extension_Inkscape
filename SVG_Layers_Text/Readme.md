## layer_text_SVG.cpp

**compile:**
g++ layer_text_SVG.cpp -o layer_text_SVG

**execute:**
./layer_text_SVG

## Usage	

**Inkscape:**
1. Open svgOut_layer.svg
2. Save as svgOut_layer.xcf

**GIMP:**
1. Open svgOut_layer.xcf
2. Export as svgOut_layer.gif
   - Mark "As animation"
   - Mark "Loop forever"
   - Mark "Use delay entered above for all frames"
